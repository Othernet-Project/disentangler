Introduction
============

Simple dependency tree builder, handles forward and reverse dependencies,
returning an ordered tree with it's nodes ordered from root towards leafs.

Source code
-----------

Confloader source code can be found `on GitHub
<https://github.com/Outernet-Project/disentangler>`_ and is released under GPLv3
license. See the ``COPYING`` file in the source tree for more information.
